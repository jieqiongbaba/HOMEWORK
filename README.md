# 一级标题：对信息素养含义的理解

​               信息素养包括：*信息意识*，*信息常识，信息能力，信息道德。*

​      信息意识：使用信息解决工作和生活问题。

​      信息常识：信息科学技术常识，让学生阅读有关信息文章，了解信息技术发展与应用。

​      信息能力：在信息系统的基本操作能力，信息的采集、传输、加工处理和应用能力，信息系统的分析和查错能力和对信息系统与信息进行评价的能力。

​       信息道德：培养学生具有正确的信息伦理道德素养，让学生能够遵循信息应用人员的伦理道德规范。
